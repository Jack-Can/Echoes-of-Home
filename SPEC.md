# 回声驿站 - 产品规格说明书

## 1. 作品名称

家的回声（回声驿站）

## 2. 作品简介

「家的回声」是一款以聊天界面为载体的亲情传递应用，旨在帮助年轻人与长辈之间建立更好的情感连接。产品分为子女端和父母端双角色，子女可以输入日常碎片信息，AI帮助生成温馨家书；父母端则提供语音输入和沉浸式阅读体验。内置日历模块，以可视化方式记录互动历程，展示关系气象，温馨时刻可被标记收藏，年末生成温情报告。核心愿景是「把你说不出口的话，翻译成家人听得懂的话」。

## 3. 技术说明

本项目使用 HTML5 / CSS3 / JavaScript 原生开发，纯前端无需服务器，双击 index.html 即可在浏览器运行。

AI 情感翻译模块使用 HuggingFace transformers.js 库（通过 CDN 引入），在浏览器端纯本地运行轻量级情感分类模型，无需后端 API。

日历模块采用以下技术实现：
- SVG 内联绘制折线图、温度计等可视化组件
- CSS Grid 实现月历布局
- CSS columns 实现瀑布流
- CSS scroll-snap 实现年度报告全屏翻页
- JavaScript 动态样式注入
- IntersectionObserver 检测滚动动画触发点

模拟数据覆盖 2026 年 1-4 月约 120 天聊天记录。

## 4. 功能模块说明

### 4.1 主聊天界面

支持子女端与父母端身份一键切换。左侧栏显示身份切换按钮、天气状态卡片、今日锦囊。右侧主聊天区域以气泡流展示历史消息，支持「聊天」和「家书」两种消息类型。聊天消息以普通气泡展示，家书消息以精美信封卡片展示，点击可展开复古卷轴阅读模式。

### 4.2 附件功能

聊天支持发送图片附件，功能特性如下：

**文件选择**：点击「回形针」按钮触发文件选择器，支持选择图片文件。非图片文件会弹出提示。

**确认发送**：选择文件后弹出确认对话框，显示图片预览、文件名和文件大小。确认后添加到聊天记录。

**存储方式**：
- Chrome/Edge（File System Access API 支持）：保存到 `assets/images` 目录
- Firefox/Safari：触发浏览器下载功能

**显示方式**：图片以缩略图气泡展示，点击可打开 lightbox 全屏查看。父母端（elder-mode）图片尺寸相应放大。

### 4.3 AI 情感翻译

情感翻译功能帮助弥合代际情感鸿沟，让双方都能理解彼此言行背后的真实情感。

**双向翻译模型**：
- **孩子看父母**：父母的消息（如叮嘱、命令、警告）→ 翻译为背后的关心意图
- **父母看孩子**：孩子的消息（如抱怨、烦躁）→ 翻译为情绪诉求和安慰建议

**技术实现**：
- **ML模型**：使用 transformers.js 加载本地 ONNX 模型（`voidful/albert_chinese_small_sentiment`，int8量化 ~12MB），在浏览器端纯本地运行
- **规则引擎**：结合精细规则模板进行 6 分类（关心/担心/疲惫/压力/需要安慰/日常）
- **温情检测**：ML模型正向情绪分数 > 0.7 时自动判定为潜在温情消息

**温情自动标记**：聊天消息和日历时间线中，AI自动检测温情消息（positive score > 0.7），显示柔和提示 ✨，点击即可快速标记（推荐标签）。

**显示方式**：消息加载时实时显示原文与翻译对照（并排布局），消息气泡下方附加淡紫色背景的翻译卡片，包含情感标签（如"💡 担心"）、原文解读和温柔表达建议。

**情感类型覆盖**：关心、担心、疲惫、压力、需要安慰、日常六类。

### 4.4 写信模态框

提供「写信」按钮打开写信模态框，支持一键成书功能。用户输入文字后点击「一键成书」（模拟 AI 整理），可将碎片文字整理为正式家书。支持添加图片（最多 9 张）。整理完成后可「装入信封」，再点击「寄给对方」，家书即添加到聊天记录中。信封送达时有精美动画效果。

### 4.4 日历模块

日历模块是本产品的核心特色模块，通过左侧栏底部的 tab 切换访问，包含以下 7 个子功能：

#### 4.4.1 温情日历视图
月历网格布局，使用 CSS Grid 实现 7 列日历。日期格子中用不同颜色圆点和 emoji 标注当天互动类型：
- 🟠 橙色圆点 → 有文字聊天
- 🟢 绿色圆点 → 有语音消息
- ❤️ 粉色心形 → 有温情标记
- 🎂 蛋糕图标 → 当天是纪念日

支持月份切换，今天日期高亮显示。点击日期可展开当日详情。

#### 4.4.2 互动时间线
点击日历某天后，在下方展开日详情面板。面板显示当日互动摘要。时间线以左右气泡形式展示父母与子女的聊天记录。支持四种消息卡片：
- 文字消息卡片
- 语音消息卡片（带播放进度条）
- 照片消息卡片
- AI 翻译卡片（粉色边框突出）

每条消息可点击「标记温情」按钮添加标签。

#### 4.4.3 温情时刻标记与收藏
消息标记功能：点击「❤️ 标记温情」可添加 #思念、#叮嘱、#开心、#牵挂、#感恩、#陪伴 六种标签。标记成功触发爱心飘散动画。支持 AI 模拟自动识别温情关键词并提示用户��记。

#### 4.4.4 温情收藏册
汇聚所有温情时刻的瀑布流页面。顶部固定筛选栏，支持按标签、时间（最近一周/一月/全部）、发送方（父母/子女/双方）筛选。点击「共读」进入全屏沉浸阅读模式，模拟双方同步阅读，配有 typewriter 文字效果和温柔提示文案。

#### 4.4.5 关系气象站与情感温度计
日历顶部显示天气图标（☀️晴天/⛅多云/🌧️雨天），根据近 3 天互动天数和消息情感计算。点击展开天气卡片，显示 7 天互动折线图（SVG 绘制）。温度计图标显示当月情感温度（0-100），根据互动天数、温情标记数、正向词汇占比计算。雨天时自动弹出互动提醒和快捷回复按钮。

#### 4.4.6 年度温情报告
点击底部按钮进入 6 屏全屏滚动报告。使用 CSS scroll-snap 实现翻页效果。包含：
- 开场（聊天天数，数字 countUp 动画）
- 最热闹的一天
- 最温情时刻
- 高频词 TOP5（词云效果）
- 全年气象回顾
- 年度寄语

#### 4.4.7 家庭纪念日与智能提醒
日历中用特殊边框和 emoji 标注纪念日（生日、节日）。临近 3 天时顶部弹出提醒条，提供 AI 建议面板。AI 建议包括礼物推荐和三种风格问候语（温暖型/幽默型/正式型），可一键切换。底部显示离家/回家倒计时卡片。

### 4.5 家书阅读器

点击聊天记录中的信封卡片后，展开家书阅读器。使用复古卷轴动画效果，模拟拆开信封的过程。信封展开后显示信纸内容，大字体显示便于阅读。底部显示寄信人信息和时间戳。支持图片附件展示。

## 5. 文件结构说明

### 根目录文件

- `index.html`：主入口页面，包含侧边栏（含导航 tab）、主聊天区、日历容器
- `SPEC.md`：产品规格说明书（本文档）

### CSS 样式文件

- `css/layout.css`：全局布局、色彩系统、重置样式、导航样式
- `css/sidebar.css`：左侧栏样式（身份切换、天气卡片、锦囊、导航 tab）
- `css/chat.css`：聊天消息气泡样式、信封卡片样式
- `css/input.css`：输入区域样式、写信模态框样式
- `css/utilities.css`：工具类、动画关键帧
- `css/calendar.css`：日历模块完整样式（色彩系统、布局、动画）

### JavaScript 文件

- `js/data.js`：主聊天模块模拟历史数据
- `js/emotion-intelligence.js`：情感智能引擎（transformers.js 分类器 + 规则模板）
- `js/app.js`：主应用逻辑（身份切换、消息发送、AI 情感翻译、写信模态框、家书阅读器、视图切换）
- `js/calendar-data.js`：日历模块模拟数据（120天聊天记录、纪念日、记忆钩子、AI建议模板）
- `js/calendar-view.js`：月历网格渲染、日期单元格、事件标记、月份切换
- `js/calendar-timeline.js`：互动时间线面板、消息卡片、AI翻译卡片、标记功能
- `js/calendar-warmth.js`：温情标记、爱心动画、收藏册瀑布流、共读时刻
- `js/calendar-weather.js`：关系气象站、情感温度计 SVG、互动提醒、快捷回复
- `js/calendar-memo.js`：家庭纪念日、AI建议面板、倒计时
- `js/calendar-report.js`：年度温情报告 6 屏、词云、countUp 动画
- `js/calendar.js`：日历模块主控，初始化、事件总线

### 资源目录

- `images/`：图片资源目录（当前为空）
- `模块设计/`：模块设计规范文档

## 6. 运行方式

双击 index.html 直接在浏览器打开，推荐使用 Chrome / Edge 浏览器。无需服务器，不支持 IE 浏览器。项目为纯前端静态文件，修改 CSS/JS 后刷新页面即可看到效果。
